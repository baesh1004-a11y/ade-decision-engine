# V4 · PC 통합 추천 워크벤치

## 목적

추천 생성, 추천 근거 비교, 추천 검증, 주문 연결을 27~32인치 PC 화면에서 한 번에 확인할 수 있도록 통합했습니다.

## 주요 화면

- 상단: 추천 수, 평균 유사도, 매수 검토, 관찰, 주문 대기, 최근 실행
- 왼쪽: 최신 추천 종목 목록
- 중앙: 현재 120일과 과거 급등직전 120일 비교, STO 흐름
- 오른쪽: 시장·업종·위험 검증과 주문관리 연결
- 하단: 과거 급등 사례와 추천 생성 당시 저장된 근거

## 원칙

- 추천 순위는 추천 생성 엔진의 `주봉 60% + STO 40%` 결과를 그대로 사용합니다.
- 워크벤치는 별도의 종합점수나 신뢰도 점수를 만들지 않습니다.
- 추천 검증은 시장·업종·위험을 설명하며 추천 순위를 변경하지 않습니다.
- 실제 주문은 기존 Trading Desk에서 사용자 승인 후 수행합니다.

## 메뉴 구조

1. 홈
2. 추천 워크벤치
3. 포트폴리오
4. 시스템

세부 기존 화면은 추천 워크벤치 그룹 아래에 유지하여 운영과 점검이 가능하도록 했습니다.
