# ADE V1 — Meta Score 통합판단 버전

백업 브랜치: `backup/v1-meta-score`
기준 커밋: `1c2d50c6f528cdff029e69e0c2a2b7285f1d9c29`

## 목적

추천 결과를 여러 점수로 다시 합산해 하나의 Meta Score로 만들고, 그 결과를 주문관리로 전달하는 버전입니다.

## 처리 흐름

1. 급등직전 120일 패턴 추천 생성
2. AI 레이더로 시장·업종 상태 계산
3. 패턴 유사도, 레이더, 시장, 업종, 위험 항목을 고정 가중치로 합산
4. `FINAL BUY`, `BUY WATCH`, `HOLD`, `PASS` 판단 생성
5. 종합판단 결과를 주문관리로 전달

## 특징

- 종합판단 점수를 별도로 계산
- 추천 점수와 종합판단 점수가 동시에 존재
- 주문 후보는 종합판단 결과를 기준으로 선택

## 한계

- 추천 유사도가 높아도 종합점수가 낮아질 수 있어 사용자가 결과를 직관적으로 이해하기 어려움
- Prediction 값이 없는 추천에서는 점수 왜곡 가능성이 있음
- 동일 정보를 여러 단계에서 재점수화해 기능이 겹침
