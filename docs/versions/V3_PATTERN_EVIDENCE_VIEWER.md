# V3 추천 근거 비교 버전

## 현재 브랜치

`main`

## 목적

AI 패턴 랩을 별도의 판단 엔진에서 추천 결과 설명 화면으로 변경했다.

## 역할 분리

### 추천 생성

- 현재 활성 종목의 최근 120거래일과 과거 실제 급등 직전 120거래일 패턴을 비교한다.
- 주봉 유사도와 STO 유사도를 계산한다.
- 추천 순위와 상위 과거 매칭 사례를 저장한다.

### 추천 근거 비교

- 최신 추천 실행 결과를 불러온다.
- 추천 생성 당시 저장된 순위와 유사도를 그대로 표시한다.
- 현재 120일과 선택한 과거 급등직전 120일을 차트로 비교한다.
- 과거 사례의 30% 도달기간과 5·10·15·20일 상승경로를 보여준다.
- 추천 생성 당시 저장된 설명 문구를 표시한다.

## 제거한 기능

- AI 패턴 랩 자체 신뢰도 점수
- CURRENT DECISION 카드
- 추천 / 관심 관찰 재판정
- 고정 85% 통과 체크리스트
- 급등속도 가중치를 이용한 별도 판단
- 표본 수에 따른 별도 경고 판정

## 핵심 원칙

추천 근거 비교 화면은 추천 순위와 판정을 변경하지 않는다.

```text
추천 생성 = 계산과 선별
추천 근거 비교 = 저장된 결과 설명과 시각화
추천 검증 = 시장·업종·위험 확인
주문관리 = 사용자 승인과 실행
```
