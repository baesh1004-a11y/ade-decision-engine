# ADE V2 — 추천 검증 버전

적용 브랜치: `main`

## 목적

급등직전 패턴 추천점수를 시스템의 유일한 핵심 점수로 유지하고, 시장·업종·위험 정보는 별도 점수 합산 없이 검증 체크리스트로 사용합니다.

## 처리 흐름

1. 현재 종목의 최근 120거래일과 과거 실제 급등직전 120거래일 비교
2. 주봉 60% + STO 40%로 패턴 유사도 산출
3. 추천 순위 생성
4. 시장 상태, 업종 상태, 위험 상태를 독립적으로 검증
5. 검증 결과를 `매수 검토`, `관찰`, `보류`, `제외`로 표시
6. `매수 검토`와 `관찰` 종목만 주문관리로 전달

## 핵심 원칙

- 종합판단 점수를 새로 만들지 않음
- 추천점수는 패턴 유사도 하나만 유지
- 시장·업종·위험은 추천점수를 올리거나 낮추지 않음
- 검증 항목은 주문 전 확인을 위한 독립 게이트로 사용

## 화면 구성

- `한국 종합판단` → `한국 추천 검증`
- `미국 종합판단` → `미국 추천 검증`
- 평균 종합점수 제거
- 패턴 유사도, 시장 상태, 업종 상태, 위험 상태를 한 화면에 표시

## 하위 호환성

기존 `final_decisions` 테이블과 `meta_score` 필드는 유지합니다. 다만 V2에서 `meta_score`에는 별도 종합점수가 아니라 원래 패턴 유사도가 저장됩니다. 이는 기존 주문관리와 성과관리 코드가 깨지지 않도록 하기 위한 호환 조치입니다.
